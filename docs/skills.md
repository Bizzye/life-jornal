## 📦 Instalação

```bash

# Instalar uma skill (requer gh >= 2.90.0)

gh skills install github/awesome-copilot <nome-da-skill>



# Listar skills instaladas

gh skills list



# Remover uma skill

gh skills uninstall <nome-da-skill>

```

Ou copie a pasta da skill manualmente para seu diretório local de skills.
